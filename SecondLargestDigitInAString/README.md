# Second Largest Digit in a String - Day 2

## Problem
Given a string `s`, return the second largest numerical digit that appears in the string. If it does not exist, return `-1`.

## Approach
- Traverse the string character by character.
- Check whether each character is a digit.
- Keep track of the largest and second largest digits found.
- Return the second largest digit. If it does not exist, return `-1`.

## Time Complexity
- O(n)

## Space Complexity
- O(1)

## Language
Java
