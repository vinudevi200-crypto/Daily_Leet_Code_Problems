# Length of Last Word

## Problem
Given a string `s` consisting of words and spaces, return the length of the last word in the string.

## Approach
- Start from the last character of the string.
- Skip any trailing spaces.
- Count the characters until a space or the beginning of the string is reached.
- Return the count.

## Time Complexity
O(n)

## Space Complexity
O(1)

## Language
Java
