# Remove Element

## Problem
Given an integer array `nums` and an integer `val`, remove all occurrences of `val` in-place and return the number of remaining elements.

## Approach
- Traverse the array using a pointer.
- If the current element is not equal to `val`, place it at the current valid position.
- Increment the valid position pointer.
- Return the number of remaining elements.

## Time Complexity
- O(n)

## Space Complexity
- O(1)

## Language
Java
